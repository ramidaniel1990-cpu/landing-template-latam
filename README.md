# Landing Page Template LATAM

Professional landing page template for small businesses in Latin America (restaurants, clinics, workshops).

## Tech Stack

- Next.js 15 + TypeScript
- Tailwind CSS
- App Router

## Architecture

- `src/types/index.ts` — all interfaces centralized here
- `src/data/config.ts` — the only file that changes per client
